# Task Manager Web Application

A modern, responsive task management web application built with vanilla JavaScript, HTML, and CSS.

## Features

- ✅ Add new tasks
- ✅ Mark tasks as completed
- ✅ Delete tasks
- ✅ Persistent storage using localStorage
- ✅ Responsive design for mobile and desktop
- ✅ Accessible with semantic HTML and ARIA attributes
- ✅ Modern CSS with gradients and animations

## Technologies Used

- HTML5 (Semantic markup)
- CSS3 (Modern styling with Flexbox and Grid)
- JavaScript (ES6+ features, DOM manipulation, localStorage)

## Getting Started

1. Clone or download the repository
2. Open `index.html` in your web browser
3. Start managing your tasks!

## Usage

- Enter a task in the input field and click "Add Task" or press Enter
- Check the checkbox to mark a task as completed
- Click the "Delete" button to remove a task
- Tasks are automatically saved to your browser's local storage

## Accessibility

This application follows web accessibility guidelines:
- Semantic HTML elements
- Proper form labels and ARIA attributes
- Keyboard navigation support
- Screen reader friendly

## Responsive Design

The app is fully responsive and works well on:
- Desktop computers
- Tablets
- Mobile phones

## Browser Support

Works in all modern browsers that support ES6 JavaScript and localStorage.
